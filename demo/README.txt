This directory contains demonstration of the various usages of Y++.

Yttrius demonstrations are rare as there are simply no mainstream
usage for it besides creating new syntax.

Exon demonstrations are not included due to its deprecation as a 
low level abstraction of Yttrius. However, it is now considered
a "library" within Y++: Y++ code can call __exon__??() to run 
exon snippets.

demos include snippets of ths Y++ standard the latest being 
iteration number 2 of 2022
